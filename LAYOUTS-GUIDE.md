## How to Use Different Layouts

In your post's front matter, simply change the `layout` field:

```yaml
---
layout: post-magazine    # For magazine style
# OR
layout: post-minimal     # For minimalist style
# OR
layout: post-visual      # For visual/grid style
# OR
layout: post-asymmetric  # For asymmetric style
---
```
